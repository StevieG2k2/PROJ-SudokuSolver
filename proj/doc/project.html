SudokuSolver is a comparison of sudoku solving algorithms that use strategies, ordered in different ways, and a backtracking algoritm. 
The strategies are written based on the description from SudokuWiki and use a Pencil-and-Paper approach. This means each empty cell of the Sudoku board will store possible number candidates that can be the solution for that cell and the solving strategies will mostly eliminate candidates.
The backtracking algoritm was written based on a video and moddified to our project structure.
