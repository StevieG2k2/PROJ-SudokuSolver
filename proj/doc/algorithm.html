The sudoku solving algortihm solves the given sudoku puzzle represented in a string of 81 characters which is then inserted into our data structure.
Input:
A string of 81 characters with "." representing empty cells
Output:
A string of 81 characters
Test of correctness:    
The result of the algorithm is correct if all the sudoku rules apply:
  • all cells are solved
  • all cells hold exactly 1 number
  • in all the rows, columns and 3x3 boxes the numbers 1 to 9 appear exactly once
