The test cases used for the sudoku solving algorithms are sudoku puzzles of various difficulty. The puzzles are passed to the algorithm in a string format of 81 characters where numbers 1-9 represent the keys of the puzzle and "." represent the empty cells.

Example: 6.......2.9.4...5...1...7...5..84.......2.......3.5.4.2.....6...3...9.8...7.....1
